# Sieve branding assets

Visual assets shipped with the open-source distribution: logos,
social cards, and CLI demo GIFs.

## Logos

| File | Use |
|---|---|
| `sieve-icon.svg` | Square mark for app icons, favicons, OS tiles |
| `sieve-icon-favicon.svg` | Simplified favicon variant |
| `sieve-wordmark.svg` | Text-only mark |
| `sieve-lockup.svg` | Full horizontal lockup (icon + wordmark) |
| `sieve-lockup-dark.svg` / `sieve-lockup-light.svg` | Theme-specific lockups used by the README |

See [`BRAND.md`](BRAND.md) for usage rules — colour palette, clear
space, minimum sizes, what not to do.

## Social card

| File | Use |
|---|---|
| `social-card-clean.svg` | Editable source |
| `social-card-clean-1200x630.png` | OpenGraph / Twitter card (1200×630) |

## CLI demo GIFs

Three GIFs, three stories. All 1200×700, Catppuccin Mocha.

| GIF | Story |
|---|---|
| `sieve-quick-install.gif` | One-command install. `pipx install llm-sieve && sieve-install`, ending on the green "Sieve is ready" panel. |
| `sieve-wizard.gif` | Interactive management menu. Navigates Service → Status, Store → Last 10 facts, back to the top menu showing state. |
| `sieve-benchmark.gif` | Benchmark replay: 15-turn medium fixture → 90% token reduction, ending on the hero "Sieve sent 90% fewer tokens" banner. |

All three are embedded in the project's [README](../README.md).
